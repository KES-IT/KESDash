# HomeDash
HomeDash 是一个基于 [Next.js](https://nextjs.org/) 和 [Semi-design](https://semi.design/zh-CN/start) 的仪表盘。
![screen-shot-one](/shot.png)
![screen-shot-two](/shot2.png)
## 填入配置文件
    
```bash
cp .env.local.example .env.local
```

## 安装依赖

```bash
pnpm install
```


## 运行前端

```bash
pnpm dev
```
